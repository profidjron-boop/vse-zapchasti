# UI Direction — Все запчасти

**UI profile:** premium-dark-saas  
**Visual mode (public):** enabled  
**Runtime policy:** NO CDN, все ассеты self-hosted.

## Brand tone
- Уверенно и по делу, без “воды”.
- Акцент на надежности, скорости подбора, гарантиях, удобной записи на ремонт.
- Короткие фразы, понятные CTA.

## Homepage must-have (MVP запрещён — флоу полноценные)
1) Hero: H1 + подзаголовок + 2 CTA
   - CTA1: “Подобрать запчасть”
   - CTA2: “Записаться на ремонт”
2) Поиск/подбор: строка поиска + базовые фильтры (марка/модель/категория) + подсказки.
3) Блок “Категории запчастей” (плитки).
4) Блок “Запись на ремонт”:
   - выбор: легковая / грузовая
   - услуги/вид работ (минимум: ТО, диагностика, ходовая, электрика, тормоза, шиномонтаж, прочее)
   - дата/время (если без расписания — заявка менеджеру)
5) Блок “Почему мы” (3–6 преимуществ)
6) Блок “Как мы работаем” (шаги)
7) Блок доверия: гарантии/сертификация/опыт/отзывы (по факту)
8) Контакты + карта (без внешних скриптов по умолчанию; карта как ссылка/скриншот-изображение self-hosted до ADR)
9) Footer: реквизиты/политики/навигация.

## Interaction & motion
- Hover/press состояния для всех интерактивов.
- Sticky top-nav на desktop, bottom-sheet на mobile при записи.

## Typography & spacing
- Крупный hero (но без “кричащего”).
- Большие поля, аккуратные разделители, ровные отступы.
