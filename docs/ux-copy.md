# UX Copy — Все запчасти

## Тон
- Коротко, уверенно, по делу.
- Без “мы лучшие”, вместо этого — конкретика: сроки, гарантия, процесс.

## CTA (примеры)
- Primary: «Подобрать запчасть», «Записаться на ремонт», «Оставить заявку»
- Secondary: «Уточнить совместимость», «Получить консультацию»

## Формы
- Лейблы понятные: «Марка», «Модель», «Год», «VIN (если есть)», «Телефон», «Комментарий».
- Подсказки: короткие, 1 строка.
- Валидация: сразу и у поля; общий алерт сверху — только если нужно.

## Ошибки
- Сообщение для человека + что сделать дальше.
- Технические детали не показывать. Если есть trace/request id: «Код: XXXX».

## Empty states
- Объяснить почему пусто и что сделать:
  - «Ничего не найдено. Попробуйте изменить запрос или выберите категорию.»

## Loading
- Скелетоны для списков/карточек.
- “Бесконечные” спиннеры не использовать без текста, если >2с.
